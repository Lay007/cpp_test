# Локальный снимок производительности

Оригинал на английском: [perf_report.md](perf_report.md)

Дата: 2026-04-17  
Платформа: Windows (MSVC, Release)

Исходный отчёт бенчмарка был сгенерирован командой:

```bash
cmake --build build --config Release --target benchmark_report
```

Результаты:

| Бенчмарк | Отсчётов | Среднее время (мс) | Пропускная способность (MSa/s) |
|---|---:|---:|---:|
| FIR 127-tap convolution | 32768 | 2.794 | 11.727 |
| Goertzel power detector | 8192 | 0.044 | 184.380 |
| GCC-PHAT delay estimate | 1024 | 179.096 | 0.006 |

Примечания:
- Эти значения зависят от железа, компилятора и конфигурации сборки.
- В GCC-PHAT сейчас используется наивная DFT-реализация, она оставлена как целевая зона оптимизации.
